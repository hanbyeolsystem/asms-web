엔지니어 매뉴얼

1. 신규 접수

  • 상단 메뉴의 [신규접수] 클릭
  • 고객 정보 / 주소 / 상품정보 / 시리얼 / 요청사항 입력
  • [접수 등록] 버튼으로 저장

2. 상태 변경

  • 접수내역 → 접수번호 클릭 → 현재 상태 라디오버튼 변경 → [저장]
  • 상태: 접수 진행 센터 견적 택배 완료 출고

3. SMS 발송

  • 접수 상세에서 [SMS 발송] 또는 메뉴의 [SMS관리] 사용
  • 잔액 부족 시 발송 실패 → 충전 후 재발송

4. 견적서 인쇄

  • 접수 상세 → [견적서 인쇄] → 브라우저 인쇄 대화상자에서 출력
